# Grafo en C

Implementación básica de un grafo no dirigido utilizando listas enlazadas y generación de matriz de adyacencia.

## Funcionalidades

- Agregar vértices
- Agregar aristas
- Contar vértices
- Generar matriz de adyacencia
- Liberar memoria dinámica

## Compilación

gcc main.c grafo.c -o grafo
./grafo

## Conceptos utilizados

- Estructuras de datos
- Listas enlazadas
- Memoria dinámica (malloc / free)
- Matriz dinámica
